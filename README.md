## 🚘 Car Information Fields

Each vehicle record contains:

- 🚗 Car Name  
- 🔢 Number Plate  
- ⚙️ Model  
- 👤 Customer Name  
- 📞 Phone Number  
- 💰 Price (Including VAT)  
- 💵 Transfer Fee  
- 🚙 Transfer Status (Yes / No)  
- ❗ Reason (if Transfer = No)
