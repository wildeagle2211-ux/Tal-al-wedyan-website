# 🚗 Tal Al Wadyan Trading – Vehicle Management System

A modern, lightweight web-based vehicle trading and management system built using HTML, CSS, and JavaScript.

This project demonstrates a simple admin dashboard for managing car records with a clean and responsive interface.

---

## 📌 Project Overview

This application allows users to:
- Login with a simple authentication system (demo)
- Add vehicle records with details
- View all stored car entries dynamically
- Delete existing records
- Manage customer-related data
